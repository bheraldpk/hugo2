    Overview of Applications List Screen #

    The Applications list screen is where you can see the list of all of the apps that have been created on your tenant.

    Viewing the Applications List #

    • Select Applications from the sidebar. The list of apps displays. From here you can edit an existing app or start a new app.

    Viewing Application Details #

    1. Select Applications from the sidebar.

    2. Click to select the app you want to view in the list. You can see the app’s information and any dependancies that exist. App dependencies allow the data model to be shared between different apps. An example of this would be a Case Management app that depends on a Patient Management app.

      You can also access the Data Schema, Business Logic, APIs, and App UI for the app.
