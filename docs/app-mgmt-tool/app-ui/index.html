    Application UI Screen Tool #

    The App UI screen is where you can create, edit, and delete UI pages.

    Managing the Apps List #

    Describe Pages List

    Viewing the Page List for an App #

    1. Select Applications from the sidebar.
    2. Select the app you want to edit.
    3. Click the APP UI button near the top of the screen. This opens the Page List showing each Page Name, Path, and Page icon for each page that has been created for the app. From the page, you can Preview the app, Create a new page, and Edit a page in the app.

    Viewing the Compoments on an App Page #

    1. Open the App UI page for the app you want to edit (for details, see Viewing Page List for an App.
    2. Select the Page you want to view.
    3. Select the Components tab and then click the componenent you want to view. The Component detail page displays, including all of the properties you can control. Editable properties are displayed with an Edit icon next to the Property Name.

    Sorting #

    To sort a column, click the column heading.

    Filtering #

    Click the FILTERS button. (Add xref to Filtering topic?)

    What is a page? #

    A page is a unique display used for different purposes, such as changing the view depending on the device (web, mobile, tablet) or creating different views for different audiences, such as managers and staff.

    A page is made up components and is managed by rules.

    Technically, a page is a component that gets mounted when a subpath matches the Page subpath configuration at the Appâ€™s Page Routing Engine.

    An app can have multiple pages, but one page can only be part of one app (as it might depend on its context). A page can render multiple Components, and one Component can be rendered by multiple Pages.

    What are components? #

    Components are presentational or data-wired UI Components/elements/functions. Components render design elements on the page to create the overall functuinality and display. A Page typically made up of an assembly of components.

    Adding a Page #

    1. Open the App UI page for the app you want to edit (for details, see Viewing Page List for an App.
    2. Click the + CREATE button in the top-right corner of the screen.
    3. In the drawer that opens on the right side of the screen, enter the follow details and then click SAVE.
      • Name- enter a name for the page
      • Path- this is generated from the name and is not editable
      • Page Icon- select an icon

    Editing Page Information #

    1. Open the App UI page for the app you want to edit (for details, see Viewing Page List for an App.
    2. Select the page you want to edit.
    3. Click the EDIT button near the top-right corner of the screen.
    4. In the drawer that opens on the right, edit page information and then click SAVE.

    Editing Page Components #

    1. Open the App UI page for the app you want to edit (for details, see Viewing Page List for an App.
    2. Select the page you want to edit.
    3. Select the Components tab and then click the componenent you want to edit. The Component detail page displays, including all of the properties you can control. You can:
      • Edit component information (click EDIT)
      • Edit component properties - editable properties are displayed with an Edit icon next to the Property Name.

    Adding Page Components #

    1. Open the App UI page for the app you want to edit (for details, see Viewing Page List for an App.

    2. Select the page you want to edit.

    3. Select the Components tab and then click the ADD button.

    4. In the drawer that opens on the right, complete the following details and then click SAVE.

      • Type- Dropdown with the list of available componentes
      • Display Name- not editable
      • Parent- Dropdown with the list of previously added components
      • Description
